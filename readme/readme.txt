Plugin for CudaText.
Gives commands to colorize text fragments (usually words) with N styles (backgnd color + italic/bold/crossed).
Changes selected text, without selection uses word under first caret.

Gives menu item in "Options/ Settings-plugins" to edit config file.
Options:
- all_words: Colorize all occurrences of source fragment
- case_sens: Case sensitive search for other occurrences
- words_only: Colorize source fragment only if it is a whole word
- whole_words: Colorize only those other occurrences, which are whole words
